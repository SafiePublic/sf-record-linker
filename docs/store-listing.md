## 日本語

### 名前

SF Record Linker

### 概要（短い説明）— 132文字以内

Salesforce Lightning レコードページのリンクをワンクリックでコピー。オブジェクトごとの項目付加やカスタムフォーマット、複数タブ一括コピーに対応。

### 詳細説明

「Claude Code で Chrome 拡張を手軽に作れないかな」とお試しで作ってみた拡張機能です。

Salesforce Lightning のレコードページで拡張アイコンをクリックするだけで、レコード名のハイパーリンクをクリップボードにコピーできます。

コピーされたリンクはリッチテキスト（HTML ハイパーリンク）とプレーンテキストの両方でセットされるため、メール・チャット・ドキュメントなどにそのまま貼り付けできます。

■ 基本機能
• ワンクリックでレコード名のリンクをコピー
• レポートページにも対応
• Salesforce のレコードページ以外ではアイコンが自動で無効化

■ オプション機能（設定画面でカスタマイズ）
• オブジェクト名の付加（例: 「商品: Product A」）
• リンク範囲の切り替え（レコード名のみ / 全体）
• 簡易設定 — オブジェクトごとに項目を1つ指定して付加
• カスタムフォーマット — テンプレート変数（${name}, ${object}, ${項目ラベル名}）で自由な形式を定義
• エラートーストコピー — 画面にエラーが表示されている時、リンクの後ろにエラー内容を追記
• 複数タブ一括コピー — 複数タブを選択してまとめてコピー（箇条書きスタイル選択可能）
• 設定エクスポート / インポート — JSON 形式で設定の保存・復元

■ プライバシー
• 個人情報の収集は一切ありません
• データの外部送信は一切ありません
• すべての処理はブラウザ内で完結します

■ 対応言語
• 日本語
• English

## English

### Name

SF Record Linker

### Summary (short description) — up to 132 characters

Copy Salesforce Lightning record page links with one click. Supports per-object fields, custom formats, and multi-tab bulk copy.

### Detailed Description

A Chrome extension casually built to see how easily Claude Code can handle it.

Click the extension icon on any Salesforce Lightning record page to instantly copy a hyperlink to the record.

The copied link includes both rich text (HTML hyperlink) and plain text, so you can paste it directly into emails, chat messages, documents, and more.

■ Core Features
• One-click copy of record name links
• Works on report pages too
• Extension icon is automatically disabled outside Salesforce record pages

■ Optional Features (customizable in settings)
• Prepend object name (e.g., "Product: Product A")
• Link scope toggle (record name only / full text)
• Simple format — add one field per object
• Custom format — define any format with template variables (${name}, ${object}, ${Field Label})
• Error toast copy — append error messages shown on page to the copied link
• Multi-tab bulk copy — select multiple tabs and copy all at once (bullet list style selectable)
• Settings export / import — save and restore settings as JSON

■ Privacy
• No personal data is collected
• No data is transmitted externally
• All processing happens entirely in your browser

■ Languages
• 日本語 (Japanese)
• English
