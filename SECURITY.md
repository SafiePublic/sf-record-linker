# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it via [GitHub Issues](https://github.com/SafiePublic/sf-record-linker/issues).

This is a personal experiment project, so there is no dedicated security team. However, we will address reported issues as promptly as possible.

---

## セキュリティポリシー

本プロジェクトでセキュリティ上の脆弱性を発見した場合は、[GitHub Issues](https://github.com/SafiePublic/sf-record-linker/issues) から報告してください。

個人の実験プロジェクトのため専任のセキュリティチームはありませんが、できる限り迅速に対応します。
